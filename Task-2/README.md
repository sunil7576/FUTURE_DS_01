# Task 2 - Customer Retention & Churn Analysis

## Objective
Analyze customer churn behavior and identify factors that influence customer retention using the Telco Customer Churn dataset.

## Dataset
- Dataset: Telco Customer Churn Dataset
- File Used: WA_Fn-UseC_-Telco-Customer-Churn.csv

## Tools Used
- Power BI
- CSV Dataset
- Data Visualization

## Dashboard Metrics
- Total Customers
- Churn Customers
- Churn Rate (%)
- Average Monthly Charges

## Visualizations

### Churn by Contract Type
Analyzes customer churn across different contract categories.

### Churn by Internet Service
Shows churn distribution among internet service types.

### Churn by Gender
Compares churn behavior between male and female customers.

### Churn by Payment Method
Identifies payment methods associated with higher churn.

### Churn by Senior Citizen
Analyzes churn patterns among senior citizen customers.

### Tenure vs Churn
Shows the relationship between customer tenure and churn.

## Filters (Slicers)
- Gender
- Contract Type
- Internet Service

## Key Insights

- Month-to-Month contract customers have the highest churn rate.
- Fiber Optic users show higher churn compared to other internet services.
- Customer churn is relatively balanced across genders.
- Customers with shorter tenure are more likely to churn.
- Long-term contracts improve customer retention.

## Recommendations

1. Encourage customers to switch from month-to-month contracts to long-term plans.
2. Improve service quality for Fiber Optic customers.
3. Introduce loyalty programs for new customers.
4. Monitor high-risk customer segments proactively.
